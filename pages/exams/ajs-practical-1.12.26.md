2. Complete the `LimitedTextarea` component as outlined in the video.

3. Apply the following styling to the component:

```javascript
// for your textarea element:
const textareaStyle = {
	width: '300px',
	height: '150px',
}

// for the paragraph element that displays the character count:
const pStyle = {
	fontFamily: 'monospace',
	fontSize: '14px',
	textAlign: 'right',
	marginTop: '4px',
}
```
